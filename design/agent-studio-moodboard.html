Agent Studio, not another dashboard.

A private encrypted studio where agent-created Markdown becomes staged, inspectable, and human-editable. We keep the Notion calm, add Raycast tactility, Linear precision, Miro collaboration, and a tiny cast of memorable agent personas.

Notion / warm canvas

Paper-like surface, quiet typography, low chrome. The main artifact should feel readable and owned.

Linear / precision

Dark metadata surfaces, translucent borders, precise density. Use for review rails and timelines.

Raycast / tactile power

Inset keycaps, command palette depth, macOS-native surface highlights. Great for studio controls.

Miro + Superhuman / warmth

Collaborative color and premium restraint. Persona color should be tiny and intentional.

Design principles

  • The document is the hero; chrome is a studio toolbench.
  • Depth comes from layered paper, inset rails, tactile controls.
  • Persona color is seasoning, not wallpaper.
  • Trust/security is always visible but quiet.
  • Fewer icons; more typography and named product surfaces.

Build next

  • Create `StudioShell`, `MarkdownStage`, `AgentBench`, `ProposalSlip`.
  • Keep Radix/shadcn primitives under custom product components.
  • Prototype Quiet Studio in the Next app before deeper editor work.
  • Test real agent Markdown, long reports, proposals, and empty/error states.
